第十一课 语法

  1. 名は　名が　一形/二形 です。

    • 这里的形容词使用表示情感的形容词，第一个名词表示情感的主体，第二个名词表示情感的对象。

    • 例：

      小野さんは　歌が　すきです。

      林さんは　犬が　怖いです。

      王さんは　映画が　嫌いです。

  2. 名は　名が　分かります。

    • 这里的使用用于描述能力的词，类似的还有：できます、上手です、下手です、苦手です。

    • 例：

      スミスさんは　韓国語が　分かります。

      私は　コンピューターが　できます。

      吉田さんは　ゲームが　下手です。

    • です变成ではありません、分かります变成分かりません　也可用于表示否定。

  3. 名や　名

    • や和 と的用法类似，都是用于列举多个并列的物

    • と要列举出所有的项目、而や只列举其中两项

    • や 常与など连用，意为 呀什么的

    • 例：

      田中さんは　時々　中国や　韓国へ　いきます。

      （言下之意是田中还会去其他国家）

      私は　お店で　野菜や　果物などを　買いました。

  4. から＆だから

    • から前的是事情的原因，から可以理解为“因为”

    • だから、后面接结果，可以理解为所以

    • 一类形容词后可以直接接から，但是二类形容词不行，（二类形容词）后必须接ですから或者だから。

    • ですから是だから的礼貌说法

    • 例：

      寒いですから、窓を　閉めます。

      明日は　休みです。だから、友達と　映画を　見ます。

  5. 一些表示频率的副词

    • いつも、よく、時々、たまに、あまり、全然

    • いつも、よく 表示频率很高（总是、经常）

    • 時々、たまに 表示频率适中（偶尔）

    • あまり、全然 表示很少、几乎不，后面动词用否定形

  6. どうしてですか

    • 用于询问事由，相当于“为什么”

    • 回答时在谓语后加から

    • 例：

      A：僕は　来月　また　ここへ　来ます。

      B：どうしてですか。

      A：ここで　友達の結婚式が　ありますから。

注：

  1. ぼく别乱用

  2. ホテルで　写真展が　あります

    • 这里为什么不用に？（部屋に　机が　あります）

    • 这里的“写真展”是一个活动，更接近一个动作，所以这里用で来表示动作（活动）发生的地点。（私は　今日　コンビニで　お弁当を　買いました。）

    • 类似要用で的词还有：結婚式、展覧会、会議。

  3. いか（如何）がですか

    • どうですか的礼貌表示

  4. そうですね

    • 可以表示附和；也可以表示”让我想想“的意思。

  5. 迷います

    • 表示困惑，难以选择

    • 也可表示迷路（道に迷いました）

  6. 気に入りました

    • 表示一种突然的喜欢（之前没见过或者之前是不喜欢的）

    • 不能对长辈用

    • 持续的喜欢用好き

  7. 省略

    • 例：

      北京ダックは　おいしいですね。

      ええ、とても（美味しい）。
